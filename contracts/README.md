# Contracts

Soroban smart contracts (Rust), one crate per contract:

- `escrow/` — P2P escrow contract
- `multisig/` — Multi-signature wallet
- `gateway/` — Payment gateway (off-chain provider confirmation bridge)
- `vault/` — Savings vault with yield

**Status:** scaffolding in place, core logic in progress. See open issues labeled `smart-contracts`.
